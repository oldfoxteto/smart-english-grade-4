# 120-Day Execution Roadmap (Sprint-Ready)

## Phase 0 (Days 1-10) - Foundation
- Freeze product scope for MVP
- Finalize architecture + DB schema
- Setup monorepo or service structure
- CI/CD, environments, secrets, observability
- Arabic UX design system (RTL tokens/components)

Deliverables:
- technical baseline
- ERD v1
- API contracts v1

## Phase 1 (Days 11-35) - Learning Core MVP
- Auth + user profile + onboarding
- Placement test (initial)
- Learning path (A1 English)
- Lesson runtime (MCQ, fill, listen)
- Attempt tracking + mastery baseline
- SRS review engine

Deliverables:
- usable learning loop end-to-end

## Phase 2 (Days 36-60) - AI Personalization v1
- Knowledge tracing model v1
- Recommendation engine v1
- AI feedback in Arabic
- AI Tutor text mode (guided scenarios)
- Learner DNA initial inference

Deliverables:
- personalized next-best-lesson
- AI tutor text pilot

## Phase 3 (Days 61-85) - Gamification + Social v1
- XP, levels, streak, badges
- Daily missions + weekly challenges
- Leaderboard + season framework
- Friends + groups + challenge rooms

Deliverables:
- retention loop with social engagement

## Phase 4 (Days 86-105) - Voice + Pronunciation
- Speech recognition integration
- Pronunciation scoring
- Voice tutor (limited scenarios)
- Emotional check-in flow

Deliverables:
- speaking practice loop
- pronunciation feedback in Arabic

## Phase 5 (Days 106-120) - Greek Launch + Monetization
- Greek A1 content release
- Subscription plans + paywall
- Certificates baseline
- Admin dashboards (learning + revenue + retention)
- Beta launch and telemetry-driven fixes

Deliverables:
- bilingual MVP (EN+EL)
- paid plan ready

## Team Structure (Minimum)
- Product Manager (1)
- Tech Lead (1)
- Frontend Engineers (2)
- Backend Engineers (2)
- AI Engineer (1)
- UX/UI Designer (1)
- Linguistics Experts EN/EL (2)
- QA Engineer (1)
- Growth/Marketing (1)

## Go/No-Go Gates
- Gate 1: D7 retention >= 25%
- Gate 2: lesson completion >= 60%
- Gate 3: AI feedback helpfulness >= 4/5
- Gate 4: paid conversion >= 3% (beta cohort)

## Launch Plan
- Closed beta: 1,000 users (Arabic-speaking)
- Open beta: 10,000 users + referral system
- Regional launch: GCC + Egypt + Levant
